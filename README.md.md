\# ColorfulSigns



Allows using \& color codes on Minecraft signs.



Example:

\&1Hello \&2World





Code	Name



\&0	Black

\&1	Dark Blue

\&2	Dark Green

\&3	Dark Aqua

\&4	Dark Red

\&5	Dark Purple

\&6	Gold

\&7	Gray

68	Dark Gray

\&9	Blue

\&a	Green

\&b	Aqua

\&c	Red

\&d	Light Purple

\&e	Yellow

\&f	White



 

\&k	Obfuscated

\&l	Bold

\&m	Strikethrough	\[Java Edition only]

\&n	Underline	\[Java Edition only]

\&o	Italic

\&r	Reset
